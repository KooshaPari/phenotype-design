# Session Overview

- Goal: provide a minimal `Taskfile.yml` for the repo's common workflows.
- Scope: detect the repo's language/tooling, map `build`, `test`, `lint`, and `clean` to the existing package scripts and generated artifacts.
- Success criteria: `task build`, `task test`, `task lint`, and `task clean` all resolve to the Bun/TypeScript stack in this repository.
